# Hist-riaChatGPT
O intuito deste projeto é criar uma página na qual dá um conteúdo introdutório da criaçao do ChatGPT e percorre suas versões de motor, tais quais: GPT-3, GPT-4 e etc.
Este projeto está sendo realizado com JavaScript como linguagem de programação, auxiliando o HTML e CSS.
A página está responsiva, atendendo resoluções como a de Celulares, Tablets e Desktop.
Para rodar a página é recomendável que você tenha a extensão "Live Server" no VScode.
